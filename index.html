Flappy Bird

by: Jaiden Lau

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

Flappy Bird has always been a java game and since it was deleted in stores, i thought that maybe I could bring it back. Many of the Flappy Bird fundamentals are part of the LeagueInvaders game we made in Module 1. So I oughted to use the LeagueInvaders game as a background for the Flappy Bird game.

2. What I Learned

I learned that the game could be more detailed and a bit easier, so i added pictures and i decreased the flapping speed of the bird to make it more plausible to fit into the pipe without losing.

3. Architecture of My Game

There are a few parts to the game like the Bird, the pipes, the States, etc. The Bird has a detection box that allows it to switch states when the pipe hits. The pipes moves with the x-- and the space between the pipes is random, but the space in between is the same. when the pipe reaches x = 0, then the pipes position will be random and teleport to the other side of the screen.

4. Technical Implementation

The Pipes where one thing I was proud of. Though i really couldn't finish it without help. With Dave, he told me the basics on how the Pipes should work and the code to it. When finished, the pipes could move from right to left and repeat, randomize its position, and maintain the space n between the pipes.

5. Most Challenging Part

The most challenging part of this project is to make the position of the Pipes random, but keeping the space in between the same distance. To overcome this, the was 2 pipes that was made. One pipe would be random, while the other would fill in the remaining space, excluding the space in between the pipes.

6. Future Fixes and Updates

I would like to improve the quality of the text and the pictures as they were very blocky. I would either change the code for the bird and make it seem like it's moving or I could make my own pictures and use that as the images/text.
